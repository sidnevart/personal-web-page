# Как интегрировать портфолио в резюме

**Принцип:** резюме — это тизер (30-секундный скан), портфолио — это пруфы (демо, скриншоты, метрики).

---

## Куда вставить ссылку

**В шапке резюме (рядом с контактами):**
```
Portfolio: sidnevart.github.io/personal-web-page
```

**В секции опыта — под каждой позицией, мелким шрифтом:**
```
T-Bank | Backend Engineer | Oct 2025 — Present
... bullets ...
→ Подробнее с демо: sidnevart.github.io/personal-web-page/#experience
```

**В секции проектов — под каждым проектом:**
```
ProofForge — open-source peer accountability (Go + Next.js)
→ Демо + скриншоты: sidnevart.github.io/personal-web-page/projects/proof-forge
```

## Почему T-Bank разбит на 4 проекта на сайте

На сайте T-Bank не одна свалка "делал много всего", а **4 продукта с демо:**

1. **AI Code Review** — `.../projects/ai-code-review`
2. **AI CI Refactor Bot** — `.../projects/ai-ci-refactor`
3. **RAG Onboarding Agent** — `.../projects/rag-onboarding`
4. **AI Office (OpenClaw)** — `.../projects/ai-office`

В резюме это сворачивается в 3–4 bullets. Если рекрутер заинтересовался — он кликает и видит synthetic UI, метрики, стек.
