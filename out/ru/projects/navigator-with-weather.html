Продукт

Навигатор с погодой

Прикладной сервис, который объединяет маршрут и погодный контекст, чтобы планировать поездку точнее.

TypeScriptNext.jsWeather API
GitHub ↗

Проблема

Пользователю сложно принимать решение о поездке, если маршрут и погодные условия живут в разных местах.

Решение

  • Объединил маршрут и прогноз в один пользовательский сценарий с быстрым ответом.
  • Сделал интерфейс минимальным: только то, что нужно для решения «ехать сейчас / позже / другим маршрутом».

Моя роль

Full-stack: интеграция API, UI и сборка проекта до готового демо.

Результат

  • Получился законченный мини‑продукт с понятной ценностью и чистой реализацией.
  • Хороший демонстрационный кейс: быстро собрать полезную фичу end‑to‑end.
