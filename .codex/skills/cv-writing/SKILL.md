# CV Writing

Use this skill for resume work.

## Required reads

1. `cv-writing.skill.md`
2. `global-market-positioning.skill.md`
3. `resources.md`
4. Relevant project docs when a metric or scope claim needs verification

## Scope defaults

- Resume scope defaults to big-tech platform work plus the commercial real estate analytics project.
- Do not expand into weaker side projects unless the user asks.

## Workflow

1. Build a clean ATS-safe structure.
2. Pull the strongest verified numbers and operational outcomes.
3. Rewrite internal terms into market-readable language.
4. Keep bullets action-led and result-led.
5. Run a final market-positioning pass before shipping the text.

## Rules

- Use real numbers when available.
- If a number is missing, use honest outcome wording instead of inventing impact.
- Prefer evidence of revenue, time saved, reliability, scale, or workflow improvement.
- Keep the document readable in under 10 seconds for a recruiter and detailed enough for a hiring manager.
